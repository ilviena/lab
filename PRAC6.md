# Выполнено Ямалетдиновой И. И., гр. 432Б

### Цель практической работы:
Изучить шифрование и расшифровку YAML-файлов с использованием утилиты SOPS и алгоритма Age.

### Ход работы:
1. Установка *age* и *sops* проводилась вручную. Файлы были установлены и помещены в заранее созданную папку "*age*". 

2. Сгенерировала ключи командой `.\age-keygen.exe -o age.key`.

3. Указала *SOPS*, где лежит ключ.
   
   <img width="634" height="114" alt="image" src="https://github.com/user-attachments/assets/05915913-c44e-48e7-af1f-2b8942bbcf9a" />

4. Для шифрования был использован файл "*me.yaml*", созданный на предыдущей практике. Переместила файл в папку "*age*".

   <img width="762" height="210" alt="image" src="https://github.com/user-attachments/assets/0a667dae-222b-4f94-839a-eabebdfed0c2" />

5. Создала конфигурационный файл *SOPS*, в редакторе указала инструкции:

   <img width="540" height="312" alt="image" src="https://github.com/user-attachments/assets/a6a25156-6185-4a44-bbbd-ad169a27469c" />

6. Зашифровала свой файл командой `.\sops.exe --encrypt me.yaml > me.enc.yaml` в файл "*me.enc.yaml*".

7. Проверила, что содержимое расшифрованного файла совпадает с оригиналом. Для этого использовала команду `.\sops.exe --decrypt me.enc.yaml`.

   <img width="671" height="319" alt="image" src="https://github.com/user-attachments/assets/9e858ed3-d1e0-4d98-aec1-8cedc017c34e" />

8. Результат совпал. Расшифровка подтверждает корректность работы *SOPS* с ключом *Age*.
